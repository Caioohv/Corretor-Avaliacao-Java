# Corretor de Avaliações em Java
Projeto desenvolvido na disciplina "Projeto de Software 1", onde o IFMG fez a requisição de um programa que automatizasse as correções dos simulados.   
### Tecnologias utilizadas:
 - Java
 - PostgreSQL
 - OpenCV
  
O Projeto só pôde ser desenvolvido graças à utilização do https://github.com/m-cakir/bubble-sheet-multiple-choice-scanner
